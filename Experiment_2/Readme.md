E-Commerce Transaction Analysis Using R
Objective

To explore data distribution and variability in e-commerce transaction data using advanced visualization techniques.

Scenario

An e-commerce startup analyzes transaction data to understand customer spending behavior and to detect abnormal purchase patterns.

Tools and Libraries Used

R Programming Language

ggplot2

dplyr

gridExtra

Experimental Tasks

Visualized the distribution of transaction amounts using a histogram.

Detected abnormal transaction values using a boxplot.

Analyzed monthly sales intensity through a heatmap visualization.

Conclusion

The experiment demonstrates the effectiveness of R-based visualizations in identifying spending patterns, variability, and anomalies in e-commerce transaction data.
